# Memory Game

Memory Game is a browser card-matching game built with plain HTML, CSS, and JavaScript. The goal is to flip two cards at a time, remember their positions, find every matching pair, and clear the board.

## Live Preview

[Play Memory Game](https://liena-grytsyna.github.io/guessgame/)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [`canvas-confetti`](https://www.npmjs.com/package/canvas-confetti)

## Getting Started

Clone the repository:

```bash
git clone https://github.com/liena-grytsyna/guessgame.git
cd guessgame
```

Run the project locally:

Open `index.html` in your browser.

## Features

- 4x4 grid of emoji cards shuffled on every load.
- CSS-based flip animation for open/match states.
- Theme toggle plus a short peek button.
- Lightweight confetti effect via [`canvas-confetti`](https://www.npmjs.com/package/canvas-confetti) when the board is complete.
